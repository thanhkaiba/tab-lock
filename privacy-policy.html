Tab Lock — Privacy Policy

Last updated: June 2025

Data collection

Tab Lock does not collect, store, or transmit any personal data. No information about you, your browsing activity, or the pages you visit is ever sent anywhere.

Permissions

The extension uses the following browser permissions solely to provide its core function of blocking keyboard input on a tab:

  • activeTab / tabs — to identify which tab the user wants to lock.
  • scripting — to inject a keyboard-blocking script into the active tab.
  • host_permissions (<all_urls>) — to allow the script to run on any tab the user chooses to lock.

All processing happens locally in your browser. No data leaves your device.

Third parties

Tab Lock does not use any third-party services, analytics, or advertising.
